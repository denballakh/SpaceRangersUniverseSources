src - исходники модов
    .build - сюда сбилдится один мод, если запустить отдельный скрипт для билда мода
    .tmp - папка для временных файлов
.tmp - папка для временных файлов
.build - папка для билда, тут самая актуальная версия. Желательно иногда ее удалять
.release - папка с модами, которые ставит установщик
.update - папка с исправлениями. Получается как разница build и release
.updates - папка с разными рахивами исправлений
update.zip - запакованная папка update

Папки .tmp, .build, __pycache__, .update и файл update.zip можно удалять, ничего важного не потеряется.
